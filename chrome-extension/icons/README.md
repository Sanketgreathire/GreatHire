# Chrome Extension Icons

You need to add 3 icon files to this folder:

1. **icon16.png** (16x16 pixels)
2. **icon48.png** (48x48 pixels)
3. **icon128.png** (128x128 pixels)

## Quick Way:

1. Go to https://www.favicon-generator.org/
2. Upload any image (🎯 emoji works great)
3. Download all sizes
4. Rename and place here

Extension will work without icons but Chrome will show default icon.
